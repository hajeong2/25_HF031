#### 1-1. 프로젝트 소개
* 프로젝트 명 : 건물 균열 탐지 수직 자율주행 로봇

* 협소한 외벽 환경에서 정밀/실시간 점검 필요성에 따라 자동화 기술 도입 요구 증가
